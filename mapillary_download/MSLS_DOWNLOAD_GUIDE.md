# Mapillary SLS Dataset Download 

The `mlysls_download_script.sh` bash script automatically download and extract the **Mapillary Street Level Sequences (MSLS)** dataset.

Links are taken from [Mapillary website](https://www.mapillary.com/dataset/places)

The script downloads the following files:

| File | Description |
|------|-------------|
| `msls_images_vol_1.zip` - `msls_images_vol_6.zip` | Six volumes containing the actual street-level images |
| `msls_metadata.zip` | Metadata files for the image dataset |
| `msls_patch_v1.1.zip` | A patch/update file (version 1.1) |
